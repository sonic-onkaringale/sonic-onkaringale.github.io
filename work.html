Footwear Application : Freelance Project


Description

This was a freelance Android project with singular requirement of making its products searchable in English as well as Marathi.
This project was built on top of View based & Composed based UI Toolkits.
It uses ObjectBox which is a High-performance NoSQL database and faster than Room Database.
It utilizes Kotlin Coroutines to manage asynchronous tasks.


Features/Keypoints

  • Uses Amazon S3 for saving server cost as the client's use-case was limited to the internal operations.
  • All Product metadata is serialized into one Json file.
  • Saves the metadata and images of the product separately in S3.
  • Intelligently handles synchronizations.
  • Uses Material3 Design and Relay for Figma.
  • It is Adaptive, layout changes based on the screen space available to it.

Multiple Linear Regression Application : Personal Project


Description

Multiple Linear Regression Android App!
This app allows you to train a multiple linear regression model and make predictions on the go using an algorithm written from scratch in C++ and interacting through the Java Native Interface (JNI).
Multiple linear regression is a statistical method used to model the linear relationship between a dependent variable and one or more independent variables.
With this app, you can easily input your data and let the app handle the rest, providing you with accurate predictions in real-time.
Whether you're a student learning about multiple linear regression for the first time, or a professional looking for a quick and efficient tool for your analysis, this app has you covered.


Features/Keypoints

  • Takes input as CSV file.
  • Automatically drops the columns with non-numerical values
  • Friendly UI.
  • Gives UI for predicting Trained Model.
